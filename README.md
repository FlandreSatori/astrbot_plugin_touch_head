# touch_head - 摸头杀插件

AstrBot 摸头杀插件 - 当用户拍机器人时自动生成并发送摸头GIF动图。

fork 自 https://github.com/mjy1113451/touch_head

## 功能特性
- 🖼️ **动态生成**: 使用用户QQ头像实时生成个性化摸头GIF
- ⌨️ **手动触发**: 支持通过 `摸摸@用户` 或 `摸摸` 消息触发
- ⚙️ **运行时参数**: 支持在 `摸摸` 指令后附加参数（qq/x/y/i/s）
- ✨ **精美动画**: 20帧流畅动画，包含完整手部和挤压效果 

## 依赖

- Pillow >= 10.0.0
- aiohttp >= 3.8.0
### 触发
发送命令：
```
摸摸 @用户
摸摸
摸摸 12345678
```

摸摸 x10 y-10 i0.02 s1.5
摸摸 12345678 x10 y-10 i0.02 s1.5
### 配置命令
在webui或机器人管理员发送消息:
### 参数说明
- `qq号`：可选，指定要摸的用户。
- `x`/`y`：可选，位置增量参数，如 `x10 y-10`。
- `i`：可选，帧间隔，范围 `0.02 ~ 1.0`。
- `s`：可选，缩放倍率，范围 `0.3 ~ 3.0`。

说明：
- 所有长期配置只能在 WebUI 中修改。
- `x` 和 `y` 是在 WebUI 配置的基础上叠加，例如基础 `x10 y10`，消息带 `x10 y-10`，最终为 `x20 y0`。
- `squish` 不支持通过消息修改。



## 技术实现

- 使用 Pillow 库本地生成GIF动画
- 通过 QQ 官方头像 API 获取用户头像
- 监听 OneBot v11 协议的 `notice.notify.poke` 事件
- 使用 base64 编码发送图片，兼容性更好
